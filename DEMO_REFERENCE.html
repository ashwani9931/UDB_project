🎯 User Workflow

  1. Open http://localhost:5000
  2. See professional landing page with CourseHub logo
  3. Click "User Login" button
  4. Enter credentials: user@coursehub.com / password123
  5. Access user dashboard with available courses
  6. Search for courses or enroll in available ones
  7. View statistics (enrolled, completed, in progress)
  8. Logout when done

⚙️ Admin Workflow

  1. Open http://localhost:5000
  2. Click "Admin" button in navbar
  3. Enter credentials: admin / admin123
  4. Access admin dashboard
  5. Add new courses using the form
  6. View all courses and manage them
  7. Check enrollment reports and analytics
  8. Monitor student statistics
  9. Logout when done
